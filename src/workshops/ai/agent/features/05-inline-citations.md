# Feature: Inline Citations

## Overview
Add citation markers ([1], [2], etc.) in message text that link to source documents/entities in the ontology.

## Key Requirements
- Parse citation markers from LLM output
- Store sources in message metadata
- Link to ontology entities (contracts, documents)
- Hover tooltip with preview
- Sources list at bottom of message

## User Story
**As a** user **I want** to see where information came from **So that** I can verify facts and explore sources
