# Feature: Image Support

## Overview
Display images in messages and support image upload/generation via S3 and Bedrock.

## Key Requirements
- Display image message parts inline
- Lazy loading for performance
- S3 storage for uploaded images
- Image generation tools (Bedrock Stability AI)
- Image analysis in context

## User Story
**As a** user **I want** to share and generate images **So that** I can work visually with the agent
