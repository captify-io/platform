# Feature: Visual Workflow Canvas

## Overview
React Flow-based visual workflow builder for creating multi-phase agent workflows.

## Key Requirements
- Drag-and-drop phase nodes
- Connect phases with transition tools
- Configure system prompt per phase
- Assign tools per phase
- Workflow progress indicator in chat
- Enhanced phase transitions with `prepareStep`
- Save/load workflow definitions

## User Story
**As a** user **I want** to design workflows visually **So that** I can create complex multi-step agent processes
