# Feature: Agent Studio Application

## Overview
Complete agent configuration and testing application at `/agentstudio` based on ontology builder.

## Key Requirements
- Agent list (grid/list toggle, filter, search)
- Agent form with tabs (basic, model, prompt, tools, workflow, memory, security)
- Model selector (provider + model picker)
- Tool selector integration (from Phase 5)
- Workflow builder integration (from Phase 6)
- Live testing panel (split view: config left, chat right)
- Agent templates library
- Clone/import/export functionality

## User Story
**As a** user **I want** a visual studio to configure agents **So that** I can create and test agents quickly
