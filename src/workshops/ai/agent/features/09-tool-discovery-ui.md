# Feature: Enhanced Tool Selector UI

## Overview
Visual tool selector with category-based browsing, search, and filtering by ontology domain.

## Key Requirements
- Category browsing (Ontology Tools, Custom Tools, Workflow Tools, Widget Tools)
- Search by name/description
- Filter by domain, status, category
- Tool preview on hover (shows schema)
- Bulk enable/disable by category
- Dependency visualization

## User Story
**As a** user **I want** to browse available tools by category **So that** I can find and enable the tools I need
