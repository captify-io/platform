# Feature: Ontology-to-Tool Auto-Generation

## Overview
Automatically generate CRUD tools from ontology nodes, including widgets as UI rendering tools.

## Key Requirements
- Generate create/get/update/delete/query tools for each ontology node
- Generate relationship query tools from ontology edges
- Include widgets (capture/display) as tools
- Zod schemas from JSON schemas
- 5-minute cache matching ontology
- Tool registry integration

## User Story
**As a** developer **I want** tools generated automatically **So that** I don't have to manually create CRUD operations
